💰 Controle Financeiro em Python

Sistema simples de controle financeiro desenvolvido em Python, com foco em praticar lógica de programação, manipulação de arquivos e organização de dados.

📌 Sobre o projeto

O Controle Financeiro em Python é uma aplicação de terminal que permite cadastrar receitas e gastos, calcular o saldo atual e manter os dados salvos em um arquivo ".txt".

O projeto foi criado para praticar conceitos fundamentais da programação de forma aplicada a um problema real: o controle de dinheiro.

🚀 Funcionalidades

- Adicionar receitas
- Adicionar gastos
- Visualizar saldo atual
- Identificar se o saldo está positivo, zerado ou negativo
- Salvar os dados em arquivo ".txt"
- Carregar os dados salvos ao iniciar o programa
- Apagar receitas cadastradas
- Apagar gastos cadastrados

🛠️ Tecnologias utilizadas

- GitHub

▶️ Como executar

1. Clone este repositório:

git clone https://github.com/m4rcusgollub/controle-financeiro-python.git

2. Acesse a pasta do projeto:

cd controle-financeiro-python

3. Execute o arquivo principal:

python main.py

💻 Exemplo de uso

======CONTROLE DE GASTOS======
1 - Adicionar receita
2 - Adicionar gasto
3 - Ver saldo
4 - Apagar receita
5 - Apagar gasto
0 - Sair

Exemplo de saldo:

R$ 70.0
Saldo positivo

📄 Armazenamento dos dados

Os dados são armazenados no arquivo "dados.txt", seguindo o formato:

RECEITA - 100.0
GASTO - 30.0

Ao iniciar o programa, o sistema lê esse arquivo e carrega automaticamente as receitas e gastos já cadastrados.
